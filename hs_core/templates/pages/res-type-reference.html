HydroShare Resource Types Reference

HydroShare is designed to help researchers and scientists share and collaborate on water data and models of all kinds. Because there are so many kinds of data and models, we have many kinds of resources to house them. In HydroShare, we define a resource as digital content which includes the science data files or model files and their corresponding metadata information. HydroShare will include multiple resource types such as time series, geographic feature class, geographic raster, multidimensional space time array data (NetCDF), model, document, etc. Currently, HydroShare beta system supports resource types of time series and geographic feature class (or Geoanalytics).

More on this can be found at our support page.
